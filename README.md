# 🚀 Task Manager App

✨ A modern full-stack MERN Task Manager application with authentication, role-based access control, and task management features.

---

## 🌟 Features

✅ User Registration & Login  
✅ JWT Authentication  
✅ Role-Based Access Control (Admin/User)  
✅ Admin Dashboard  

---

## 🛠️ Tech Stack

### 🎨 Frontend
- React.js
- React Router DOM
- Axios
- CSS

### ⚙️ Backend
- Node.js
- Express.js
- MongoDB
- JWT Authentication

---
### 🌍 Deployment

🚀 Backend → Render
💾 Database → MongoDB Atlas
🌐 Frontend → Vercel
